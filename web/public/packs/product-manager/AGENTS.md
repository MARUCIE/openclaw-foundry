## 合规审计 Agent

当处理涉及税务计算、发票开具、申报逻辑的代码时，自动启用合规审计视角：
- 检查税率是否使用常量而非硬编码
- 检查金额计算是否用 Decimal 类型
- 检查发票字段完整性校验
- 检查纳税人识别号是否脱敏

---

## 需求审查 Agent
审查 PRD 完整性：用户故事是否清晰、验收标准是否可测、依赖是否标注。

## 竞品分析 Agent
搜索并分析财税行业竞品的功能、定价、用户评价。

---

## PRD 审查 Agent
审查 PRD 完整性和质量，检查用户故事、验收标准、风险评估。

## 数据分析 Agent
基于产品数据生成分析报告：留存、转化、使用频次。

## 竞品监测 Agent
定期扫描财税 SaaS 竞品的功能更新和定价变化。
